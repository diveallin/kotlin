### 상속한 클래스의 구현을 by 로 넘겨준 인스턴스에 위임

```kotlin
interface Base {
    fun print()
}
class BaseImpl(val x: Int) : Base {
    override fun print() { println(x) }
}

// "by b" 가 없다면 Derived 는 print()를 구현하지 않았기 때문에 오류!
// b 로 위임
class Derived(b: Base) : Base by b

fun main() {
    val b = BaseImpl(10)
    Derived(b).print() // 10 출력
}
```
